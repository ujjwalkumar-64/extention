About PageGenie

PageGenie brings AI-powered reading and writing tools to any web page. It prefers fast, private on-device AI when available and seamlessly falls back to your backend for advanced features.

  • Selection toolbar: Summarize, Explain, Rewrite, Translate, Save
  • Quick fixes: Proofread replacement, Translation overlay, Insert code comments
  • Modes: Auto, Offline only, Online only
  • “Quiz Me” and Curated Reading via backend
